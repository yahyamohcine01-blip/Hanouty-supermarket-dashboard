Hanouty Supermarket Dashboard

Overview
This repository contains the first open, publicly shared dataset of Moroccan supermarket chains and an interactive Power BI dashboard built from it.

The goal is to provide a starting point for analysis of the modern retail landscape in Morocco and to showcase data analytics and dashboarding skills using Power BI.

Project Story
The dataset was web‑scraped from publicly available online sources and cleaned using Power Query.

A Power BI report was then created from scratch, including data modelling, DAX measures, and visual design tailored to the Moroccan market.

Repository Structure
Moroccan_Supermarkets_Dashboard.pbix – main Power BI report file.

data/ – folder containing the underlying dataset(s) used to build the dashboard (CSV format).

Dataset
The dataset includes store‑level information for Moroccan supermarket chains, such as chain name, city, and other descriptive attributes useful for mapping and comparison.

It is intended for learning, experimentation, and exploratory analysis, and can be extended with new variables or chains over time.

Dashboard Features
The Power BI dashboard allows users to explore:

Geographic distribution of supermarket stores across Moroccan cities and regions.

Comparison between different chains using simple KPIs and visuals.

Users can download the .pbix file and open it in Power BI Desktop to interact with all report pages and modify the analysis.

How to Use
Clone or download this repository.

Open Moroccan_Supermarkets_Dashboard.pbix in Power BI Desktop.

If needed, point the report to the dataset in the data/ folder using Power Query data source settings.
